# Word to PDF Converter

A React Native mobile application built with Expo that converts various file types to PDF format.

## Features

- Upload .doc, .docx, .pdf, .jpg, .jpeg, .png, and .mp4 files
- Select images from gallery or capture with camera
- One-click conversion to PDF
- Batch file conversion support
- Cross-platform support (iOS and Android)

## Technologies Used

- React Native
- Expo
- expo-document-picker
- expo-file-system
- expo-print
- expo-image-picker
- expo-sharing
- expo-media-library
- expo-video-thumbnails

## Installation

1. Install dependencies: `npm install`
2. Start the development server: `npm start` or `expo start`

## Usage

1. Launch the app
2. Select files using the available options
3. Tap "Convert" to convert files to PDF
4. Save the converted PDFs to your device
